# Cadastro de Pessoas - Java

Projeto de CRUD em Java via terminal, desenvolvido para praticar fundamentos de Programação Orientada a Objetos.

## Funcionalidades
- Cadastro de pessoas
- Listagem de registros
- Busca por nome
- Atualização de dados
- Remoção de registros

## Estrutura do projeto
- `Pessoa` → modelo de dados
- `CadastroPessoas` → regras de negócio e manipulação da lista
- `Main` → fluxo do sistema e interação com o usuário

## Conceitos praticados
- Programação Orientada a Objetos
- Encapsulamento
- ArrayList
- Estruturas de controle (`while`, `for`, `switch`)
- Validação de entrada com `Scanner`
- Operações de CRUD

## Como executar
1. Compile os arquivos `.java`
2. Execute a classe `Main`
3. Use o menu no terminal para interagir com o sistema

## Objetivo
Consolidar a base em Java antes de avançar para projetos com APIs e conceitos mais avançados de POO.
